How they are learned best

1. Hands-on first (very important)

Open the software and:

Cut clips 剪片

Add text 加文字

Drop transitions 拖转场

Break things 把东西搞坏

Undo a lot 😄 疯狂撤销

For Premiere (editing 剪辑)

Timeline logic (tracks, layers) 时间线的逻辑（轨道、层级）

Cuts vs transitions 剪切 vs 转场

Audio basics (levels, dialogue vs music) 音频基础（音量、对白 vs 背景音乐）

Basic color correction 基础调色


For After Effects (motion graphics / VFX) （动效 / 特效）

Layers + keyframes 图层 + 关键帧

Parent & child relationships 父子关系

Masks & mattes 蒙版和遮罩

Easing (graph editor later, not day one) 缓动（曲线编辑器可以以后再碰，不是第一天）

Pre-comps (this one clicks only after using it) 预合成（这个一定要用过几次才会真正“懂”）


A good learning order (no suffering version)

Phase 1: Copy & play

Follow short tutorials

Recreate effects exactly

Don’t ask “why” too much yet

Phase 2: Modify

Change colors

Change timing

Break animations on purpose

Phase 3: Understand when stuck

Now you look up:

“Why does my keyframe feel stiff?” 僵硬的

“What is precomp actually doing?”

“Why does AE feel slow?”

The one exception

If you plan to do advanced AE (expressions, complex rigs): （表达式、复杂绑定）

Then yes, some logical thinking helps

But you can delay that way later

Most motion designers never touch expressions beyond copy-paste.

Computer science: understand → then build

AE / Premiere: build → get confused → understand → build better
